# Implementation-of-an-18-Bit-ADC-Data-Acquisition-System-with-AXI-DMA-on-ZedBoard-
High-speed data acquisition system using the LTC2338-18 18-bit ADC with a Zynq-7000 FPGA. The design features a complete SPI interface with FIFO buffering and DMA (Direct Memory Access) for efficient data transfer to DDR memory without CPU intervention.
